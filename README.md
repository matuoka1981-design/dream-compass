# 🧭 Dream Compass

**あなたの夢を叶える人間関係ナビ**

LINEトーク履歴をAIが分析し、チーム編成・コミュニケーション戦略・目標達成ロードマップを提案するWebアプリ。

## 機能
- 🌌 人間関係マップ（星座ビジュアライゼーション）
- 👥 人物別の関係性分析
- ⚡ 目標に合わせたチーム編成提案
- 💬 人別コミュニケーション戦略
- ✉️ 仲間への誘い方テンプレート
- 🗺️ 目標達成ロードマップ
- 🪞 自己分析（レーダーチャート）
- 🧭 AI相談モード

## セットアップ

```bash
npm install
npm run dev
```
